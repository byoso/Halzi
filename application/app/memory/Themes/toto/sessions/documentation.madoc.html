Assistant: Voici un exemple simple de Gtk.Button avec un style CSS personnalisé :

import gi
gi.require_version('Gtk', '3.0')
from gi.repository import Gtk, Gdk

# Création du bouton
button = Gtk.Button(label="Cliquez ici")

# Définition du style CSS
css = """
.my-button {
    background-color: #ff6b6b;  /* Rose */
    color: white;
    border-radius: 8px;
    padding: 10px 20px;
}
"""

# Création du fournisseur CSS
provider = Gtk.CssProvider()
provider.load_from_data(css.encode())

# Ajout du style au contexte de l'écran
screen = Gdk.Screen.get_default()
Gtk.StyleContext.add_provider_for_screen(
    screen,
    provider,
    Gtk.STYLE_PROVIDER_PRIORITY_APPLICATION
)

# Application de la classe au bouton
button.get_style_context().add_class("my-button")

# Création de la fenêtre
window = Gtk.Window(title="Exemple CSS")
window.set_default_size(200, 100)
window.add(button)

# Affichage de la fenêtre
window.show_all()
Gtk.main()

Explication clé :
1. Le style CSS est défini dans une chaîne (css) avec la classe .my-button
2. Le Gtk.CssProvider charge le style
3. Le style est ajouté au contexte de l’écran avec priorité élevée
4. La classe my-button est appliquée au bouton via add_class()

Résultat :
Un bouton rose avec du texte blanc, des coins arrondis et un padding.
Le style s’applique à tous les widgets ayant la classe my-button.

Assistant: Voici un exemple complet avec Glade :

📁 Fichier example.glade

<?xml version="1.0"?>
<interface>
  <object class="GtkWindow" id="window">
    <property name="title">Exemple Glade</property>
    <property name="default-width">300</property>
    <property name="default-height">200</property>
    <child>
      <object class="GtkButton" id="my_button">
        <property name="label">Cliquez ici</property>
        <property name="css-classes">my-button</property> <!-- Classe CSS ici -->
      </object>
    </child>
  </object>
</interface>

📜 Code Python (main.py)

import gi
gi.require_version('Gtk', '3.0')
from gi.repository import Gtk, Gdk

# CSS personnalisé
css = """
.my-button {
    background-color: #4ECDC4;  /* Cyan */
    color: white;
    border-radius: 8px;
    padding: 10px 20px;
}
"""

# Création du fournisseur CSS
provider = Gtk.CssProvider()
provider.load_from_data(css.encode())

# Application du style
screen = Gdk.Screen.get_default()
Gtk.StyleContext.add_provider_for_screen(
    screen,
    provider,
    Gtk.STYLE_PROVIDER_PRIORITY_APPLICATION
)

# Chargement de l'interface Glade
builder = Gtk.Builder()
builder.add_from_file("example.glade")

# Récupération du widget
window = builder.get_object("window")
button = builder.get_object("my_button")

# Affichage
window.show_all()
Gtk.main()

🔍 Points clés

  1. Classe CSS dans Glade :
    - La propriété css-classes du bouton lie directement la classe .my-button au widget.
    - Aucun appel à add_class() en Python n’est nécessaire.

  2. Priorité du style :
    - Le fournisseur CSS est ajouté avant l’affichage pour garantir l’application du style.

  3. Fichier Glade :
    - Vous pouvez éditer le fichier .glade avec un éditeur comme glade ou gedit (via le menu “XML” dans l’éditeur).

📌 Résultat

Un bouton cyan avec du texte blanc, des coins arrondis et un padding.
Le style est appliqué via le fichier Glade et le CSS personnalisé.
